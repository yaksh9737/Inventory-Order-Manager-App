- Track product history
- Dashboard with analytics
- Responsive design with Tailwind CSS
- manage all product inventory order details stock details and manage all user and distributer order and stock

## Prerequisites

- Node.js
- npm or yarn
- MongoDB
- Tinymce

## Installation

1. **Clone the repository:**
   ```bash
   git clone https://github.com/your-username/your-repo.git
   cd your-repo
   ```

2. **Backend Setup:**
   ```bash
   cd Backend
   npm install
   ```

3. **Frontend Setup:**
   ```bash
   cd ../tea_client
   npm install
   ```
4. **admin Setup:**
   ```bash
   cd ../tea_admin
   npm install
   ```

## Running the Application

1. **Run Backend:**
   ```bash
   cd server
   npm start
   ```

2. **Run Frontend:**
   ```bash
   cd ../tea_client
   npm start
   ```
 3. **Run admin panel:**
   ```bash
   cd ../tea_admin
   npm start
   ```
